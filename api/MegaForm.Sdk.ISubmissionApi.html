SubmitAsync(int, Dictionary<string, object>, MegaFormScope?, CancellationToken)

Submit form data, running the same server-side validation as a public form submit. When the host registered the full submission pipeline (anti-spam, notifications, workflow), this delegates to it; otherwise it validates and inserts the row. On a validation failure the result's Success is false and no row is saved.
