## API Contract (all via FormController, working)
- Form/Get?formId=X — get form + schema
- Form/Save — save form + schema  
- Form/ListAll — list all forms
- Form/Stats?formId=X — submission count
- Form/Duplicate?formId=X — clone form
- Form/ModuleConfig?moduleId=X — get module config (was ModuleConfig/Get)
- Form/SaveModuleConfig — save config (was ModuleConfig/Save)
- Submit/Post — submit form data
- Submissions/List?formId=X — list submissions
- Submissions/Get?submissionId=X — get single submission
- Submissions/UpdateStatus — update status
- Submissions/Export?formId=X — export CSV
